## Project Overview

This repository contains all the work done during my **SIWES II** industrial training on **Database Application Programming in Java**.  

The project demonstrates:
- Design and implementation of a relational database using **MariaDB** (MySQL compatible)
- Proper SQL scripting following the official MySQL Courseware (Bin Binary Technologies)
- Version-controlled development using Git
- Preparation for Java JDBC integration (desktop application development)

The database models a simple **Student Payment & Records Management System** suitable for Nigerian universities.
